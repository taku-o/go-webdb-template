## About This Project

This project provides a production-ready template for building scalable web applications with:

- **Go API Server**: High-performance backend with Echo framework
- **Next.js Frontend**: Modern React-based frontend
- **Database Sharding**: Horizontal scaling with 32-table partitions across 4 databases
- **Auth0 Integration**: Secure authentication and authorization
- **Redis Integration**: Rate limiting and job queue support
