# cc-sdd

## 概要
使用駆動開発を行うためのツール。
Cursor上で動作する。

## コマンド
```
## プロジェクトの解析
/kiro:steering

## プロジェクトの要件・設計・タスクを作成(一括)
/kiro/spec-init "{プロジェクトの要件}"

## プロジェクトの要件を作成
/kiro:spec-requirements "{プロジェクトの要件}"

## 設計を作成
/kiro:spec-design "{追加の情報}"

## タスク一覧を作成
/kiro:spec-tasks "{追加の情報}"

## 指定タスクを実行する
/kiro:spec-impl "{タスク}"
```
